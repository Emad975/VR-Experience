# أسلحة النماذج (glTF/.glb)

ضع هنا مجسّمات الأسلحة الواقعية ليستخدمها مختبر الرماية
[`military-shooting-3d.html`](../military-shooting-3d.html):

```
models/m9.glb        ← مسدس M9
models/m16a2.glb     ← بندقية M16A2
```

## مهم
- التحميل يتم عبر **خادم** (مثل GitHub Pages الذي تستخدمه). متصفّحات سطح المكتب
  تحجب تحميل `.glb` محلياً عبر `file://` لأسباب أمنية — هذا متوقّع.
- إن لم يُعثر على الملف، يستخدم المختبر **نموذجاً بديلاً مُعايَراً** من أشكال Three.js
  (مناظره الحديدية وظيفية)، فيظلّ التصويب صحيحاً والمختبر يعمل.

## المعايرة بعد إضافة النموذج
لكل سلاح كتلة `CALIB` داخل ملف المختبر تحدّد — بإحداثيات النموذج المحلية —
المواضع التي تربط المجسّم بالميكانيكا:

| الحقل | المعنى |
|------|--------|
| `scale`, `rot` | تحجيم/تدوير المجسّم ليشير محوره الأمامي إلى `-Z` |
| `grip`     | نقطة مقبض اليد المسيطرة (تنطبق على يد التحكّم) |
| `foreGrip` | نقطة الحامل الأمامي (يد الإسناد في المسكة الثنائية) |
| `barrelTip`, `barrelDir` | طرف الماسورة واتجاه الإطلاق (خط النار) |
| `rearSight`, `frontSight` | الفرضة والشعيرة (للمحاذاة البصرية) |

عدّل هذه الأرقام مرّة واحدة لكل مجسّم جديد حتى تنطبق المناظر على خط النار.

> ملاحظة ترخيص: لديمو عسكري، استخدم مجسّمات تملك ترخيصها (ملكك أو حزمة مرخّصة).
> لا تُدرج مجسّمات مجهولة المصدر.
