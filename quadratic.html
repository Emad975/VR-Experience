المعادلة التربيعية العامة:
ax² + bx + c = 0

الحل بالقانون العام:
x = ( -b ± √Δ ) ÷ 2a   |   Δ = b² - 4ac

تفسير المميز Δ:
• Δ > 0 → جذران حقيقيان مختلفان (القطع المكافئ يقطع المحور مرتين)
• Δ = 0 → جذر حقيقي مضاعف (القطع المكافئ يماس المحور)
• Δ < 0 → لا جذور حقيقية (القطع المكافئ لا يقطع المحور)
